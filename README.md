# JNU
## excel.m

处理每周宣讲信息。

可使用`.bat`文件运行，不打开matlab
```
matlab -nojvm -nodesktop -nodisplay -r excel
```
## data_picker_91job_for_teacher.py

采集每天宣讲信息。

可使用`.bat`文件运行。
```
pythonw data_picker_91job_for_teacher.py
```

## weekly_excel.py

和上面的`excel.m`功能一致。

```
python weekly_excel -begin_date 2018-05-12 -end_date 2018-05-25
```
